# Деплой дашборда на Streamlit Cloud

Streamlit Cloud — бесплатный хостинг для Streamlit-приложений.
Дашборд будет доступен по ссылке вида: https://svetlux-tracker.streamlit.app

---

## Шаг 1. Создать аккаунт на GitHub

Если нет аккаунта: https://github.com → Sign up (бесплатно)

---

## Шаг 2. Создать репозиторий

1. На GitHub нажми **New repository**
2. Название: `svetlux-tracker`
3. Видимость: **Private** (чтобы данные не были публичными)
4. Нажми **Create repository**

---

## Шаг 3. Загрузить файлы в репозиторий

В репозиторий нужно загрузить 3 файла:
- `dashboard.py`
- `requirements.txt`
- `positions.db` (база данных с позициями)

Самый простой способ — через браузер:
1. Открой репозиторий на GitHub
2. Нажми **Add file → Upload files**
3. Перетащи все три файла
4. Нажми **Commit changes**

**Важно:** После каждого еженедельного прогона трекера нужно обновлять
`positions.db` в репозитории — загружать новую версию файла.

---

## Шаг 4. Подключить Streamlit Cloud

1. Заходи на https://share.streamlit.io
2. Нажми **Sign in with GitHub** — войди через свой GitHub-аккаунт
3. Нажми **New app**
4. Выбери:
   - Repository: `svetlux-tracker`
   - Branch: `main`
   - Main file path: `dashboard.py`
5. Нажми **Deploy**

Через 1–2 минуты дашборд будет доступен по ссылке.

---

## Доступ для сотрудника

Streamlit Cloud для private-репозиториев поддерживает два варианта доступа:

**Вариант А (проще):** Сделать репозиторий публичным →
дашборд доступен по ссылке без пароля.
Данные о позициях при этом видны всем по ссылке.

**Вариант Б (с защитой):** В настройках приложения на Streamlit Cloud
раздел **Sharing** → добавить email сотрудника.
Сотрудник войдёт через свой Google/GitHub аккаунт.

---

## Обновление данных (еженедельно)

После каждого прогона rank_tracker.py:

1. Открой репозиторий на GitHub
2. **Add file → Upload files**
3. Загрузи новый `positions.db` (перезапишет старый)
4. **Commit changes**

Дашборд обновится автоматически в течение минуты.

---

## Локальный запуск (без деплоя)

Для проверки на своём компьютере:

```
pip install streamlit plotly
streamlit run dashboard.py
```

Откроется браузер с дашбордом на http://localhost:8501
